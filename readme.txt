== Description ==

This plugins allows you to add a post image grid anywhere within a post or a page. The grid is responsive created using a shortcode.

Usage: Specify the ids of the posts you want to add in the image grid in the short code. Example: [postimagegrid posts="1,110,114,116,118"]
